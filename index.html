Minature Tower Crane

Retrofit an existing tower crane system to supplement future research endeavors in crane safety control research.

Main Skills Used: Solid Works, AutoCAD Eagle, Python, PyQt, Actuator Selection, Machining

What?

  • Retrofit a miniature tower crane system to utilize c++ & Arduino instead of Siemens PLC
  • System must implement model based controls to reduce payload oscillation
  • Design HMI interface in python to optimize user experience

How?

Hardware

  • Torque and speed analysis for motor and gearbox selection
  • Lath and Milling to fabricate mounting brackets
  • Utilized EAGLE to design custom Arduino shield
  • Setup CAN communication framework

Software and Human Machine Interface

  • Implemented threading to ensure processes where implement on specific intervals
  • Follows industrial crane standards
  • Implementation allows for multiple layouts depending on user
  • Implemented customer feedback to optimize user experience

Results

  • Functional system with working HMI interface for future research
  • System can record data with 10ms resolution and updates velocity every 5ms
  • Allows users to implement custom models for model based control
  • Timer Interupts allow critical systems to remain robust through student add ons
  • Can automatically implement Input Shaping feed forward control
  • Built in methods reduced payload oscillation by 95% for trolley axis
  • Ramped velocity trajectories to lower actuation effort
  • Client: Georgia Institute of Technology Crane Safety Lab
  • 2024

Robo Picasso

Create a automated system that can mimic human painting from following brush strokes to dipping the paint brush, and is able to process and run GCODE.

Main Skills Used: Solid Works, Solid Works FEA, Python, and House of Quality

What?

  • Design a device that can paint murals on various sized walls
  • Degrees of freedom should allow robot to mimic human brush strokes
  • Performed a needs analysis to initiate the design process

How?

  • Designed on Solid Works
  • Performed SolidWorks FEA to reinforce high stress areas
  • Utilized Python to perform inverse kinematics and GCODE interpretor
  • Troque calculations to determine motor selection
  • Jacobian analysis to minimize singularity conditions

Results

  • Painted a wall over 200ft wide
  • 4 degree of freedom arm to mimic human wrist movements
  • Python interpretor to interface GCODE into end point trajectories
  • Client: Senior Design and Formations Studios
  • 2023

Titan Drive Push Pull Tool

Desing a tool to push disabled 1,100 kg robotic units and pull them out from under inventory shelfs.

Main Skills Used: Solid Works, Solid Works FEA and Welding, DFM, and Mass Production

What?

  • Push disabled 1,100 kg robots across warehouse and pull from under shelf's
  • Reduce Worker Injuries from this process
  • Collaborate with Ergonomics and Production Teams

How?

  • Designed on Solid Works
  • Performed SolidWorks FEA to reinforce high stress areas
  • Utilized Solid Works Welding in main body design
  • Reinforcement brackets contain welding locating features
  • Updated design to include OTS components to minimize custom components

Results

  • Implemented DFM principles to reduce overall part mass production cost
  • Redirected push force direction to lower ergonomic load
  • Final push force was 85Ns
  • Client: Amazon Robotics
  • 2022
